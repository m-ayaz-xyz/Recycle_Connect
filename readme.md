# ♻️ Recycle Connect

A modern full-stack web application that demonstrates how users can connect with nearby recycling vendors through a location-based platform.

> **Portfolio Project**  
> This repository has been developed for learning, demonstration, and portfolio purposes.

---

## Highlights

- Secure User & Vendor Authentication
- Location-based Vendor Discovery
- Pickup Scheduling
- Order Management
- Interactive Maps
- Responsive Design

---

## Built With

- Next.js
- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- React Leaflet
- OpenStreetMap

---

## Note

Although this project includes authentication, protected routes, and various application features, it is still a portfolio project. Some features may be incomplete or may change as the project continues to evolve.
